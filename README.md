🧼 Singapore Hawker Centres Temporary Closure — ETL Pipeline & Analysis
This project builds an automated ETL pipeline to extract, clean, and transform Singapore hawker centre closure data using Python, SQL, and API integration. It normalises quarterly closure schedules into event‑level records and surfaces regional and seasonal closure patterns. This project also includes an interactive Folium map visualises closure distribution across Singapore.

📊 Key Insights

West and Central regions show higher closure frequency
Q3 has the most scheduled closures
Several hawker centres follow recurring maintenance cycles
Spatial clusters appear in mature estates
